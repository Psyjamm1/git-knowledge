# **Git knowledge**

## *Commands*

* **touch** Создание файла


* **mkdir** Создание папки


* **git init** Сделать папку репозиторием с помощью инициализации


* **git add** Добавить файлы перед сохранением командой коммит


* **git commit** Выполнить коммит
